# Developer Portfolio

A modern developer portfolio built with [Astro](https://astro.build), showcasing projects, certifications, and skills in full‑stack development and cloud operations.  
Deployed automatically with GitHub Actions to [riteshrajshetage.me](https://riteshrajshetage.me).

## Features
- Clean, recruiter‑friendly design
- Projects and certifications with visual highlights
- Responsive layout for desktop and mobile
- Automated CI/CD workflow using GitHub Actions

## Tech Stack
- **Astro** – static site generator
- **TypeScript / JavaScript** – components and scripts
- **Node.js** – runtime for builds
- **HTML5 / CSS3** – layout and styling
- **SVG assets** – logos and illustrations
- **Responsive design** – mobile‑friendly UI
- **Git / GitHub** – version control
- **GitHub Actions** – CI/CD automation
- **GitHub Pages (custom domain)** – hosting
